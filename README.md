# Event Management System

This is a simple web-based Event Management System developed using PHP and MySQL. The project allows users to view available events and register online, while the administrator can manage events, registrations, and contact messages through an admin dashboard.

## Features

- View all available events
- Register for events online
- Admin login and dashboard
- Add, edit, and delete events
- View participant registrations
- Contact form with message management
- Responsive user interface using Bootstrap

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- Bootstrap 5
- JavaScript

## Database

Import the `event_management.sql` file into phpMyAdmin before running the project.

## How to Run

1. Download or clone this repository.
2. Copy the project folder into the `htdocs` directory of XAMPP.
3. Start Apache and MySQL from the XAMPP Control Panel.
4. Import the `event_management.sql` file into phpMyAdmin.
5. Open your browser and visit:

```
http://localhost/Event-Management/
```

## Project Purpose

This project was developed as a practice/academic project to learn web development using PHP and MySQL. It demonstrates the basic functionalities of an Event Management System, including event management, participant registration, and administrator controls.
